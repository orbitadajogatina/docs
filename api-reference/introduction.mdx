Bem-vindo a primeira versão da API do bot Pergunta do Dia! É aqui que você vai encontrar a lista de rotas disponíveis na API feita para interagir com o banco de dados do bot Pergunta do Dia. Será necessário um conhecimento básico sobre requests para entender essa documentação. Se você precisa de ajuda, por favor, tire dúvidas na thread do projeto! 

### Antes de começar...

- O URL base da API é `https://pergunta-do-dia.onrender.com/api/v1`.
- Algumas rotas usam [Snowflakes](https://discord.com/developers/docs/reference#snowflakes), por conta do Discord.
- Eis o significado de cada status de perguntas:
  - 0: Aguardando revisão de um administrador;
  - 1: Recusada por um adminstrador; 
  - 2: Aprovada por um adminstrador; 
  - 3: Perguntada já enviada pelo bot.
- Para obter uma chave, acompanhe as instruções da seção Autenticação.
